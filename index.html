Création d'un fichier "ReadMe.md" pour GitHub

Cette page d'exercice explique pas à pas, les différentes étapes nécessaires pour la mise en place d'un fichier "ReadMe.md" pour le serveur de versionning GitHub.

Informations de références

Les actions suivantes ont étés effectuées grâce à l'IDE Visual Code Studio et avec des commandes "Bash" inscrites dans le terminal.

    Liste des actions :
  1. Positionnement dans le dossier de travail > Commande utilisée: "cd"
  2. Création d'un nouveau dossier "Création_ReadMe" > Commande utilisée: "mkdir Création_ReadMe"
  3. Positionnement dans le dossier "Création_ReadMe" > Commande utilisée: "cd Création_ReadMe"
  4. Création d'un nouveau fichier ".html" se nommant "Exercice-GitHub-Création_ReadMe.html" > Commande utilisée : "touch Exercice-GitHub-Création_ReadMe.html">
  5. Création d'un nouveau fichier ".md" se nommant "Exercice-GitHub-Création_ReadMe.html" > Commande utilisée : "touch Exercice-GitHub-ReadMe.md">
  6. Ajout des informations dans le fichier ".html" et dans le fichier ".md"
  7. Enregistrer l'état actuel du projet. > Commande utilisée "git add ."
  8. Commiter l'instantané de tous les changements apportés au répertoire de travail > Commande utilisée "git commit -m"
